Ny++5

Ny++5 ist ein interpretierter Pseudoassembler, welcher in Java geschrieben wurde.

Die nachfolgende Version, Ny++6, war der Reinfall, weshalb sie nicht gerne hier erwähnt wird. Sie wurde, anders als alle bisherigen Ny++-Versionen, in C++ geschrieben.

Ein Programm das zwei Zahlen addiert in Ny++5: 

{main}
    defi i, 5;
    defi i2, 2;

    say "Hallo Welt!";@
    say "Addiere 2 auf 5: ";

    add i, i2;
    prv i;@



Weiterentwickelt wird Ny++ nicht, aber eine Dokumentation ist vorhanden und Fragen werden immernoch beantwortet, sollten welche entstehen.
