TRICL

Der TRICL, a.k.a. Triploit Client, ist eine Paketverwaltung für GitHub User. Der TRICL wurde dazu entworfen, Projekte von GitHub einfach auf den Computer von Nutzern zu installieren.

TRICL wurde von SecPM abgelöst, da SecPM eine viel übersichtlichere und bessere Verwaltung hat.
